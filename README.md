# This is my emacs config

This config holds all that i like & want to use in Emacs.


Cheatsheet:
```
Magit:
C-x g: open magit-status
s: stage
u: unstage
c c: make commit
C-c C-c: commit commit
M a: add remote

Dired:
C-x d: open dired
+: add new dir
q: go back
m: select
u: unselect
U: unselect all1
d: Mark for deletion
x: execute
^: go up a dir
R: move selected to other buffer
C-x C-q: writeable mode, for renaming stuff
	     Once done, C-c C-c or C-x C-s to exit write mode
         
Commenting:
C-x C-;: Comment or uncomment a region

Centaur tabs:
C-c <left>: move to left tab
C-c <right>: move to right tab

Multi cursor:
C-<: Mark next line
C->: Mark previous line
C-c C-<mouse 1>: Mark cursor on click
C-c C-<: Mark all selected

Company:
C-Ã¤: Open company menu, if open, tobble backend

Drag stuff:
M-<up>: move selected up
M-<down>: move selected down

Emacs:
C-x C-f: open file, if no file, make file
C-x b: change buffers
C-x C-b: show all open buffers
	     d x can be used to close buffers in the list
c-x k: kill cucrrent buffer
C-x t t: open treemacs
C-c s: save & eval-buffer
C-x C-s: save-buffer
C-x C-c: close Emacs
C-c q: restart Emacs
C-c t: open terminal

Windows
C-x o: change between open
C-x 0: close current
C-x 1: open current
C-x 2: split vertical
C-x 3: split horizontal

Help:
C-h f: search functions
C-h k: search chords
C-h v: search variable
```
